    worker

        id
        md5                 -> identificacao md5
        queue_md5           -> relacao com queue

        class_name          -> nome da classe worker utilizada
        result_table_name   -> tabela onde o resultado foi armazenado
        result_table_md5    -> identificado do resultado na tabela.

        created_at      -> data de envio para queue.
        updated_at      -> data de alteracao no banco
        started_at      -> data de inicio
        finished_at     -> data de fim
        success         -> bool que indica se foi um sucesso ou nao
